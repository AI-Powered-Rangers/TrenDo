# Trend To-Do: Claude Code 개발 프롬프트

## 프로젝트 개요
**Trend To-Do**는 숏폼(틱톡·릴스·쇼츠)이 만든 유행을 전 세대가 직접 체험할 수 있게 번역하고 연결하는 문화 참여 플랫폼입니다.

핵심 철학: "보는 것에서 하는 것으로" — 수동 소비를 능동 참여로 전환

---

## 기술 스택
- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express
- **AI**: Claude API (유행 번역·분석), 이미지 생성 API
- **지도**: Kakao Map API
- **DB**: Supabase (PostgreSQL)
- **배포**: Vercel (프론트) + Railway (백엔드)

---

## 핵심 기능 6가지

### 기능 1. 숏폼 → 현실 전환 엔진 (최우선 구현)
사용자가 반복 시청한 콘텐츠를 실제 행동 가능한 챌린지로 변환합니다.

**구현 사항:**
- 사용자가 숏폼 URL 또는 키워드를 입력
- Claude API로 콘텐츠 분석 → "하고 싶은 경험" 추출
- 즉시 실행 가능한 챌린지 카드 생성 (재료 목록, 단계별 가이드, 예상 소요 시간)
- 챌린지 카드 UI: 제목 / 난이도 / 필요 재료 / 단계별 가이드 / 구매 링크

```typescript
// 챌린지 카드 데이터 구조
interface ChallengeCard {
  id: string
  title: string
  trend_source: string      // 원래 숏폼 유행
  difficulty: 'easy' | 'medium' | 'hard'
  duration_minutes: number
  materials: Material[]
  steps: Step[]
  generation_variants: GenerationVariant[]  // 세대별 버전
  local_variants: LocalVariant[]            // 지역별 버전
  traditional_connection?: string           // 연결된 전통문화
}
```

---

### 기능 2. 유행 번역 시스템
하나의 유행을 세대·지역·전통문화 맥락에 맞게 재해석합니다.

**구현 사항:**
- 같은 챌린지를 4개 세대 버전으로 자동 생성
- Claude API 프롬프트로 세대별 언어·접근법 차별화

**세대별 번역 프롬프트 예시:**
```
[10대용]: 트렌디하고 빠른 언어. "이미 알지? 더 잘 만드는 법"
[30-40대용]: 실용적이고 따뜻한 언어. "아이랑 주말에 해봐요"
[50-60대용]: 친절하고 단계적인 언어. "어렵지 않아요. 이것만 있으면 돼요"
[전 세대 공통]: "온 가족이 같이 해보는 챌린지"
```

---

### 기능 3. 지역 재해석 엔진
전국 유행을 지역 특산물·공간·전통문화와 결합합니다.

**구현 사항:**
- 사용자 위치 감지 (Geolocation API)
- 지역별 특산물 DB 연동
- Claude API로 지역 특색 결합 챌린지 자동 생성

**예시:**
```
두쫀쿠 유행 입력
→ 서울: 쑥 두쫀쿠 (인사동 쑥 카페 연결)
→ 전주: 흑임자 두쫀쿠 (전주 한옥마을 공방 연결)  
→ 제주: 한라봉 두쫀쿠 (제주 감귤 농장 연결)
→ 부산: 유자 두쫀쿠 (기장 유자 특산물 연결)
```

---

### 기능 4. 전통문화 자연 삽입
챌린지 흐름 안에 전통문화를 자연스럽게 배치합니다.

**구현 사항:**
- 전통문화 DB 구축 (음식·공예·음악·놀이 카테고리)
- Claude API로 유행-전통 연결점 자동 탐색
- 챌린지 카드 하단에 "사실 이게 OO의 현대판이에요" 섹션 표시
- 강요하지 않는 자연스러운 노출 (옵션으로 펼치기)

**예시 연결:**
```
두쫀쿠 → "찹쌀떡·약과의 현대판" 연결
버터런 → "조선시대 격구·봉쥬르 달리기 문화" 연결
봄동비빔밥 → "24절기 춘분 제철 음식 문화" 연결
야구 사진 챌린지 → "전통 탈춤 표정 연결"
```

---

### 기능 5. 문화 잔존율 (Cultural Retention Score)
조회수 대신 "현실 지속성"을 측정하는 새로운 지표입니다.

**구현 사항:**
- 챌린지 완료 체크 (D+1, D+7, D+30 팔로업)
- "한 번 해봤어요" vs "지금도 하고 있어요" 구분
- 사용자별 문화 잔존율 점수 계산
- 챌린지별 전국 평균 잔존율 공개

**측정 지표:**
```typescript
interface CulturalRetention {
  challenge_id: string
  completion_rate: number      // 챌린지 완료율
  day7_retention: number       // 7일 후 지속률
  day30_retention: number      // 30일 후 지속률  
  became_hobby: number         // 취미로 이어진 비율
  shared_with_family: number   // 가족과 함께한 비율
  retention_score: number      // 종합 잔존율 점수 (0-100)
}
```

---

### 기능 6. 문화 순환 지도 (K-Culture Flow Map)
유행이 어떤 지역·세대·전통으로 연결됐는지 실시간 시각화합니다.

**구현 사항:**
- Kakao Map 기반 전국 지도
- 챌린지 참여 핀 실시간 표시
- 유행 → 지역 → 전통 연결 흐름 라인 표시
- "두쫀쿠 챌린지: 전국 23만 가족 참여" 집계
- 시간 슬라이더로 유행 확산 과정 리플레이

## UI/UX 설계 지침

### 디자인 컨셉
- **키워드**: 활기차고 따뜻한, 세대를 아우르는, K문화스러운
- **색상**: 따뜻한 코랄+딥 네이비 조합 (K문화 에너지)
- **폰트**: 한국어 Noto Sans KR + 영문 Clash Display
- **톤**: 밝고 에너지 넘치되 모든 세대가 거부감 없이 사용 가능

### 핵심 화면 3개 (MVP)
1. **홈 피드**: 지금 뜨는 유행 카드 목록 + 세대 필터
2. **챌린지 상세**: 번역된 챌린지 카드 + 지역·전통 연결
3. **문화 지도**: 전국 참여 현황 실시간 시각화

### 세대별 UI 분기
```typescript
// 세대 설정에 따라 UI 자동 조정
const generationConfig = {
  teen: { fontSize: 'base', language: 'trendy', emoji: true },
  adult: { fontSize: 'base', language: 'friendly', emoji: true },
  senior: { fontSize: 'lg', language: 'gentle', emoji: false }
}
```

## Claude API 핵심 프롬프트

### 유행 → 챌린지 변환 프롬프트
```
당신은 대중문화 유행을 전 세대가 직접 체험할 수 있는 챌린지로 변환하는 전문가입니다.

입력된 유행: {trend_name}
사용자 세대: {generation}
사용자 지역: {region}

다음을 JSON으로 반환하세요:
1. 챌린지 제목 (세대별 언어로)
2. 필요 재료 목록 (3-5개)
3. 단계별 가이드 (3-5단계)
4. 연결되는 전통문화 (있다면)
5. 지역 특색 버전 (해당 지역 특산물 활용)
6. 난이도 (easy/medium/hard)
7. 예상 소요시간 (분)

규칙:
- 10대: 트렌디하고 빠른 언어
- 30-40대: 실용적이고 따뜻한 언어  
- 50대 이상: 친절하고 단계적인 언어
- 전통문화 연결은 강요하지 말고 자연스럽게
```

### 문화 잔존율 분석 프롬프트
```
사용자의 챌린지 참여 기록을 분석해서
이 경험이 일회성 소비로 끝났는지,
실제 생활 변화로 이어졌는지 평가하세요.

참여 기록: {participation_log}

반환:
- retention_score (0-100)
- insight (한 줄 피드백)
- next_challenge_recommendation
```

## 발표 시연 시나리오

```
1. [홈 피드] "지금 두쫀쿠가 뜨고 있어요" 카드 클릭
2. [번역] 세대 선택 → "50대 엄마용" 선택
3. [챌린지] 단계별 가이드 + "사실 이게 찹쌀떡의 현대판이에요" 전통 연결
4. [지역] "전주에서는 흑임자로 만들어요" 지역 버전
5. [완료] 챌린지 완료 체크 → 문화 잔존율 점수
6. [지도] 전국 지도에 우리 핀 찍히는 장면
```
